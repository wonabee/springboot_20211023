 - 로그인 구현
 - 어노테이션 9개
  - @Null : Null만 허용
  - @NotNull : 빈 문자열, 공백 허용, Null사용 x
  - @NotEmpty : 공백 허용, Null과 빈 문자열 x
  - @NotBlank : Null, 공백, 빈 문자열 x
  - @Email : 이메일 형식 검사. 빈 문자열 통과 (@Pattern을 통한 정규식 검사 더 많은 사용)
  - @Pattern : 정규식 검사
  - @Size : 길이를 제한할 때 사용
  - @Max : value 이하의 값만 허용
  - @Min : value 이상의 값만 허용
  - @Valid : 검증이 필요한 메서드에 설정
